# AI Usage Disclaimer

This project was built as a personal learning exercise, and AI was used throughout as a supporting tool to enhance the development, documentation, and learning process. Specifically, AI assistance contributed to:

- **Development** - explaining cloud technologies in depth, error messages, and suggesting code structures and troubleshooting approaches
- **Documentation** - drafting and refining supporting documents
- **Learning** - working through unfamiliar concepts with guidance and examples, rather than relying on trial and error alone

All AI assisted content was reviewed, and technical claims were cross-checked against official documentation where possible. The goal of using AI here was to accelerate and deepen the learning experience, not to replace the hands-on work the Cloud Resume Challenge is designed to teach. AI generated or AI assisted files are noted where relevant.
